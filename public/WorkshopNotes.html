TechBash Workshop Tips

The following notes are meant to help with the different ways you can participate in the 2024 TechBash workshop, Accessibility Testing with Screen Readers and More. Many of the links on this page will open in new tabs/windows because they go to sites external to this one.

Installing Screen Readers

Most of what I will show you, with regards to a screen reader, will apply to any screen reader. However, the keyboard shortcuts and some of the functionality will vary between them. If you use Windows or MacOS, there is a screen reader built into your operating system that does not require any additional installs. This is typically the best option if you are using a locked down computer that you cannot freely install applications on. Navigate to the Turning on you screen reader section for details on how to turn on one of those screen readers.

Alternatively, if you would like to follow along using the same screen reader I will be primarily using, and are using a Windows based computer, NVDA can be installed to a flash drive as a portable version. The portable version does not install anything to the machine itself and thus can sometimes be used unless your machine also blocks the use of flash drives.

Installing NVDA

NVDA only runs in a Windows operating system. If you are using MacOS, please the the Turning on VoiceOver section. Otherwise the installation of NVDA is pretty straight forward.

During installation there are options to enable/disable whether NVDA starts on the login screen or after you login. It is suggested that you disable the start on login screen if you share a machine. If you need to disable these settings after installation, see the Disabling the Run on Login Screen and/or Login option section.

  1. Download the NVDA installer from the NV Access website.
  2. Run the installer and follow the prompts to install NVDA.
  3. Once installed, you can run NVDA by pressing the Windows key, typing "NVDA", and pressing enter.

Installing Portable NVDA

If you are installing the portable version, the option will be one of the first ones you see when you run the installer. You need to plug in your flash drive before starting the installer. Past that the steps are pretty much the same, except you don't get as many of the customization options.

Comfiguring Narrator to start on login screen and/or at login

To turn on Narrator on Windows, perform one of the following:

  • Press Windows Key + Ctrl + Enter.
  • Press Windows Key + Ctrl + N to open the Narrator settings, and then toggle on the "Use Narrator" option.

To start Narrator automatically, you will need to go into the Narrator settings and check the box next to one, or both, of the following settings:

  • "Start Narrator after sign-in" if you want it to start after your account logs into the machine
  • "Start Narrator before sign-in" if you want Narrator to run on the login screen for all users.
