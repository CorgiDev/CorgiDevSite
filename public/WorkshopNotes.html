Screen Readers

I will be using the NVDA screen reader for my demonstration, but most of what I show will have an equivalent on other screen readers. Some keyboard shortcuts, customization options, or how things are viewed may just differ.

Some key things to note:

  • NVDA, Narrator, and JAWS are all Windows only screen readers. Voiceover is a MacOS/iOS only screen reader. TalkBack is an Android specific screen reader.
  • We will not delve into mobile testing demonstrations, but I will include some resource links in the Additional Resources section of this page.
  • NVDA can be installed as a portable version on a flashdrive or just a folder on your computer. This can often allow you to install it on a machine that doesn't allow you to install applications normally, but you don't get all the same auto-start options necessarily.

Installing Screen Readers

Narrator & Voiceover

If you use Narrator (Windows) and Voiceover (MacOS/iOS), they are built into their respective operating systems. No additional isntalls are required aside from possible voice/language packs that may be available.

If you want to install NVDA (Windows):

  1. Download the NVDA installer Opens in new window from the NV Access website.
  2. Run the installer and follow the prompts to install NVDA. Select the portable install option when it is presented if that is how you wish to install it.
    • Note that there is also an option at the start to perform a portable install if you want to install it on a flashdrive.
    • The portable version can also be used to install NVDA on some computers where installation would otherwise be blocked because it doesn't add entries to the system registry..
    • The portable version will not allow for some of the automatic startup options because it doesn't add system registry entries.
  3. Once installed, you can run NVDA via the NVDA shortcut (on the desktop or in the start menu), or by activating the main executable (exe) file.

During installation there are options to enable/disable whether NVDA starts on the login screen or after you login. It is suggested that you disable the start on login screen if you share a machine. If you need to disable these settings after installation, see the Disabling the Run on Login Screen and/or Login option section.

Installing NVDA

NVDA only runs in a Windows operating system. If you are using MacOS, please the the Turning on VoiceOver section. Otherwise the installation of NVDA is pretty straight forward.

  1. Download the NVDA installer Opens in new window from the NV Access website.
  2. Run the installer and follow the prompts to install NVDA.
  3. Once installed, you can run NVDA by pressing the Windows key, typing "NVDA", and pressing enter.

Installing Portable NVDA

If you are installing the portable version, the option will be one of the first ones you see when you run the installer. You need to plug in your flash drive before starting the installer. Past that the steps are pretty much the same, except you don't get as many of the customization options.

Comfiguring Narrator to start on login screen and/or at login

To turn on Narrator on Windows, perform one of the following:

  • Press Windows Key + Ctrl + Enter.
  • Press Windows Key + Ctrl + N to open the Narrator settings, and then toggle on the "Use Narrator" option.

To start Narrator automatically, you will need to go into the Narrator settings and check the box next to one, or both, of the following settings:

  • "Start Narrator after sign-in" if you want it to start after your account logs into the machine
  • "Start Narrator before sign-in" if you want Narrator to run on the login screen for all users.
